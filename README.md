# Console Banking System

A beginner-friendly banking application built using Core Java. The application runs in the terminal and allows users to perform basic banking operations such as checking balance, depositing money, and withdrawing money through a menu-driven interface.

---

## Features

- Check Account Balance
- Deposit Money
- Withdraw Money
- Interactive Menu
- Loop Until Exit
- Input Handling using Scanner
- Switch Statement
- Beginner Friendly

---

## Technologies Used

- Java
- Scanner Class
- Switch Statement
- While Loop

## How to Run

Clone the repository

```bash
git clone https://github.com/yourusername/Console-Banking-System.git
```

Compile

```bash
javac Main.java
```

Run

```bash
java Main
```
